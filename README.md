# 🌐 Introdução à Página Web - IFTO

Este repositório foi criado com o objetivo de armazenar e organizar todas as atividades desenvolvidas na disciplina de Desenvolvimento Web do IFTO.

## 📚 Sobre o Projeto

Neste repositório serão incluídas todas as atividades propostas pelo professor **Vinicius Rios**, com foco na aprendizagem prática de tecnologias web como:

- HTML  
- CSS  
- JavaScript  
- Estruturação de páginas web  
- Boas práticas de desenvolvimento  

## 🎯 Objetivo

O principal objetivo deste repositório é:

- Registrar a evolução do aprendizado ao longo da disciplina  
- Servir como portfólio acadêmico  
- Facilitar o acesso e revisão das atividades desenvolvidas  

## 📁 Organização

As atividades estarão organizadas em pastas, seguindo o padrão:
/atividade-01
/atividade-02
/atividade-03

Cada pasta conterá arquivos como:

- `index.html`  
- `styles.css`  
- `script.js`
